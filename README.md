# Калькулятор чаевых

Учебный веб-проект: калькулятор чаевых из урока 1. Позволяет посчитать сумму чаевых и итоговую сумму с учётом процента.

## Функции
- Ввод суммы счёта
- Ввод процента чаевых
- Отдельный вывод суммы чаевых
- Итоговая сумма с учётом чаевых
- Плейсхолдеры в полях ввода
- Зелёная кнопка "Рассчитать" с hover-эффектом

## Как запустить
1. Открыть проект в VS Code.
2. Запустить index.html через Live Server.
3. Ввести сумму счёта и процент чаевых, нажать кнопку "Рассчитать".
